**Method:** WebSearch verification of URL structure on the firm's own domain (`*.firmname.com`). WebFetch was permission-denied in this environment, so each URL was confirmed via a search-result match returning the firm-domain URL plus a content snippet describing the case. URLs are formed exactly as returned by the search engine. Items marked **VERIFIED-URL** mean the firm-domain URL was returned by search; the page should be sanity-checked on first visit before bulk download.

## Notes on App / Hidden Content Extraction

1. **No firm publishes a free mobile case-prep app.** The "Bain has an app" rumor is wrong as of 2026 — they have web-based prep, not an app. Several third-party apps (CaseTools, CaseBasix, Prepmatter) market themselves as Bain/BCG/McKinsey prep, but they are NOT firm-published.

2. **McKinsey Solve (Imbellus)** — gated behind application. The ecosystem-building game and data-analysis modules cannot be extracted without an invitation. Third-party simulation services exist but they are imitations, not the real content.

3. **BCG Casey Chatbot** — gated behind application. The 3 quiz.bcg.com cases are the only publicly-playable BCG content.

4. **Deloitte SPA extraction** — `caseinterviewprep.deloitte.com` is a JavaScript single-page-app. To extract all 13 cases for offline use: load each case in a browser, then either (a) print-to-PDF each screen, or (b) inspect network traffic in DevTools to find the JSON case payloads (look for XHR calls when navigating between cases — they likely return structured case data).

5. **Wayback Machine** — worth checking `web.archive.org` for older McKinsey cases that may have been removed (e.g., Disco-Drinks was a historical PSI case — search did not find it on current mckinsey.com, suggesting it was pulled).

## Bottom Line

**~45 distinct firm-published practice cases** publicly available across all 17 firms surveyed. Counting prep PDFs, workbooks, and practice tests as separate artifacts brings the total to **~55 unique IP-clean items**.

For a target of "1000+ cases," the firm-direct sources will only get you ~5% of the way there. The bulk of any 1000+ corpus must come from:
- MBA consulting club casebooks (Wharton, Kellogg, Booth, Tuck, Darden, Columbia, MIT Sloan, INSEAD, LBS, NYU, etc. — each publishes 20-50 cases/year)
- Public archives (PrepLounge, ManagementConsulted, IGotAnOffer free libraries, RocketBlocks, etc.)
- Historical scribd / archive.org dumps

Best high-quality firm-direct yield (rank order):
1. **McKinsey** (8 cases + 3 PSTs)
2. **Oliver Wyman** (7 cases)
3. **ZS Associates** (3 cases)
4. **BCG** (3 quizzes + 2 PDFs)
5. **Deloitte** (13 cases inside SPA — needs extraction work)
6. **Bain** (2 cases + 2 video walkthroughs)
7. Mid-tier (Kearney, Roland Berger, L.E.K., Simon-Kucher, Capgemini Invent) — 1-2 each
