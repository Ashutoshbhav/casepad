# Indian B-School Consulting Casebooks — Deep Research

**Compiled:** 2026-04-30
**Researcher:** Claude (Opus 4.7) for Ash
**Scope:** Indian PGP/MBA consulting club casebooks. Focus on direct-fetchable PDF URLs and Google Drive folders.

---

## Important methodology note (READ FIRST)

**Verification is search-snippet only.** Bash/PowerShell HTTP HEAD checks and WebFetch were both blocked in this sandboxed session, so no URL below was independently confirmed to return `application/pdf` with a 200 response. Each URL listed was returned by Google/Bing as a real result (not invented), but the actual download/access status (paywall, login wall, geo-block, Drive permission, deletion) was NOT live-tested.

Status legend used below:
- **SEARCH-CONFIRMED** = URL surfaced as a real, indexed result with a matching snippet/title
- **OFFICIAL DOMAIN** = URL is hosted on the school's own domain (`.ac.in` / `.edu`) — highest trust
- **UNVERIFIED** = URL only inferred/extrapolated; needs manual click-test before relying on
- **NO PUBLIC CASEBOOK FOUND** = exhaustive search returned no Indian-school-specific casebook

Recommendation: open the top 5-10 URLs in a browser before depending on them. Document-host platforms (Scribd, CourseHero, CliffsNotes, StudyLib) typically require a free signup; SlideShare, PDFCoffee, LinkedIn posts, and `*.ac.in` PDFs are usually fully public.

## Caveats and dead ends

1. **Live verification not done** — Bash and PowerShell denied for HTTP HEAD checks, WebFetch denied for content fetch. URLs are search-snippet-confirmed only. Always click-test before depending on a link.
2. Document-host platforms (Scribd, CourseHero, CliffsNotes, StudyLib, Studocu, Coursesidekick, PDFCoffee) usually require a free signup. SlideShare and PDFCoffee are usually fully open. LinkedIn posts require a LinkedIn login.
3. Several "official" club emails appear in the casebook copyright notices (e.g., consult@iimidr.ac.in for IIM Indore, icon@iimb.ac.in for IIMB) — direct outreach is the fallback for the 5 dead-end schools.
4. The user prompt referenced **CSCB** (IIM-B), **SAC** (ISB), **XCog/XSCC** (XLRI) — actual club names verified during research are **ICON** (IIMB), **Consulting Club** (ISB), **CRUX** (XLRI). Worth correcting in user-facing copy.
5. No evidence found of a single mega-Drive folder aggregating Indian casebooks; the closest aggregators are the LinkedIn compilation posts (Pallav Goyal, Abhishek Pandey) and the FMS Delhi Google Site archive for FMS-only.
